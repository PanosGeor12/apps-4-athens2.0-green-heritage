# GreenHeritage

GreenHeritage is a Next.js application built for the Apps4Athens Hackathon 2.0, aiming to promote sustainable tourism, cultural heritage, and green urban experiences in Athens.  

The app is intended as a foundation for building features like interactive maps, eco‑friendly routes, points of interest (POIs), events, and gamified experiences that encourage locals and visitors to explore the city responsibly.

---

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript  
- **Styling:** CSS (with support for any CSS-in-JS or utility framework you choose to add)  
- **Font:** `next/font` with Geist (Vercel’s font family)  
- **Package manager:** npm / yarn / pnpm / bun (any supported by Next.js)
